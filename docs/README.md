Progettazione e sviluppo di un database relazionale per la gestione di una catena di supermercati.

Il progetto include la modellazione E-R, la normalizzazione dello schema logico, la creazione delle tabelle, l'implementazione di chiavi primarie e chiavi esterne, viste (VIEW), trigger e transazioni SQL.

Tecnologie utilizzate:
• SQL Server
• T-SQL
• Database Relazionali
• Modellazione E-R
• Normalizzazione dei dati

Funzionalità principali:
• Gestione clienti e punti fedeltà
• Gestione prodotti e magazzino
• Gestione dipendenti e sedi
• Gestione ordini e vendite
• Monitoraggio delle scorte
• Reportistica tramite viste SQL
